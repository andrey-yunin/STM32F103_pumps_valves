* Проект по созданию ПО для биохимического анализатора ДДС 240

Данный проект содержит решения по рефакторингу платы управления насосами и клапанами на базе микроконтроллера STM32F103. 

** Необходимая документация содержится в папке readme, в составе

- план работы (файл refactoring_plan.md)
- отчет о работе проекта до рефарминга(файл project_report.md)
- общая архитектура для всех исполнителей, подключемых к дирижеру (файл executor_architecture_guide.md)
- readme/conductor_interface папка содержит файлы дирижера, необходимые для создания CAN взаимодействия.

  
 ** Правила работы: 
  - Произвести анализ проекта
  - Предоставить техничесие обоснованные решения без корректировки кода
  - Обсудить решение, возможно сделать необходимые корректировки
  - Предоставить блоки кода с комментариями
  - В большинстве случаев корректровка кода производится вручную 
  - Автоматические корректировки производить только после согласования
  - Работа происходит последовательно, каждый блок когда сопровождается моей корректировкой и подтверждением завершения.
